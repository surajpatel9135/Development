--> when we talking about development then we need know about client-server architecture.
client:- it is a computer or system or application that initiates request for user's services, data, resources, etc..
---> it is graphical interface of user's computer.

server:- server is high powered system or computer that store data, manage network resources, and give responds according to the request.
--> take request process it and send back to user browser.


[client] -------> [request] -------> [server] -------> [processes] --------> [responds] ------> [client].

--> in which one big role of internet that help to communicate or interact to each other.
--> client sends request via internet.
